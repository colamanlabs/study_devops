# IntelliJ IDEA 설정



### 1. 메뉴 한글, 영문 전환
```aiignore
1. "ctrl + alt + s" Settings 선택

2. 좌측 패널 트리메뉴에서 plugins 선택

3. 상단 탭메뉴에서 "Installed" 선택

4. IDE Localization 에서 
- 체크박스 모두 해제하면 영문모드
- "Korean Language Pack / 한국어 언어팩" 선택 
 
5. IDE 재기동 

6. "ctrl + alt + s" Settings 선택

7. Appearance & Behavior > System Settings > Langauage and Region 선택 

8. Language 를 "Korean 한국어" 로 선택

9. IDE 재기동

```



### 2. 코딩폰트 변경 (D2Coding)
```aiignore
1. "ctrl + alt + s" Settings 선택

2. 에디터 > 글꼴 선택 
- 기본값 : JetBrains Mono
- 크기 : 13.0
- 줄간격 : 1.2
- 이음자 활성화 : 비활성화 

3. 에디터 > 글꼴 선택 
- 기본값 : JetBrains Mono ( D2Coding 폰트가 잘 안맞는 것 같다. )
- 크기 : 13.0
- 줄간격 : 1.5
- 이음자 활성화 : 비활성화 
```


### 3. 마크다운 편집기 우측 미리보기창 폰트변경
```aiignore
1. "ctrl + alt + s" Settings 선택

2. "언어 및 프레임워크 > Markdown" 선택
- 기본값 미리보기 글꼴 크기 : 13.0
- 기본값 미리보기 레이아웃 : 수직으로 분할 

3. "언어 및 프레임워크 > Markdown" 선택
- 미리보기 글꼴 크기 : 16.0
- 기본값 미리보기 레이아웃 : 모니터가 작으면 "수평으로 분할 " 선택
```
