
# gitlab_ce 설치 

```
1) sudo apt update


2) sudo apt install -y curl ca-certificates tzdata openssh-server


3) curl https://packages.gitlab.com/install/repositories/gitlab/gitlab-ce/script.deb.sh | sudo bash


4) sudo EXTERNAL_URL="http://***.***.***.***" apt install gitlab-ce


5) sudo gitlab-ctl reconfigure


6) sudo cat /etc/gitlab/initial_root_password


7) sudo gitlab-ctl status


8) 방화벽 (ufw 사용시 옵션)
   sudo ufw allow OpenSSH
   sudo ufw allow http
   sudo ufw allow https
   sudo ufw reload
   
9) 웹브라우져에서 http://***.***.***.*** 접속
   
```
